マンション管理業務アプリ - 概要と操作方法
================================================================================

【アプリ概要】
このWebアプリケーションは、マンション管理の現場で発生する問題や疑問に対して、
AI（OpenAI GPT-4o-mini）が専門家の視点から回答を提供するツールです。

2つの異なる専門家モードを搭載しており、状況に応じて使い分けることができます。


【搭載されている専門家モード】

1. 管理業務主任者モード
   - 日常的な管理業務に関する実務的なアドバイスを提供
   - 現場での具体的な対応手順や注意点を提示
   - 通常業務の範囲内で対応可能な問題に適しています

2. マンション管理士モード
   - 法律的・専門的な観点からの詳細な見解を提供
   - 区分所有法や管理規約に基づいた専門的なアドバイス
   - 管理組合運営や長期的な課題への対応に適しています


【操作方法】

■ステップ1: 専門家モードの選択
  アプリ画面上部のラジオボタンで、相談したい専門家を選択します。
  - 「管理業務主任者」: 日常業務の相談
  - 「マンション管理士」: 法律・専門的な相談

■ステップ2: 問題の入力
  テキスト入力欄に、現場で発生した問題や疑問を具体的に記述します。

  【入力例】
  - 「共用部分の電球が切れているとの連絡がありました」
  - 「管理組合の総会で修繕積立金の値上げについて質問がありました」
  - 「駐車場の契約更新手続きについて教えてください」

■ステップ3: 回答の取得
  「回答を取得」ボタンをクリックすると、AIが選択した専門家の視点から
  回答を生成します。

■ステップ4: 回答の確認
  画面に表示された回答を確認し、業務に活用してください。
  必要に応じて、別のモードでも相談することができます。


【技術仕様】

使用技術:
- フロントエンド: Streamlit
- AI連携: LangChain + OpenAI API (GPT-4o-mini)
- プログラミング言語: Python 3.11

主要パッケージ:
- streamlit: Webアプリケーションフレームワーク
- langchain: LLM統合フレームワーク
- langchain-openai: OpenAI APIとの連携
- python-dotenv: 環境変数管理


【ローカル環境での起動方法】

1. 仮想環境の有効化:
   source /Users/aokitadashi/python-practice/env_practice/bin/activate

2. ディレクトリ移動:
   cd /Users/aokitadashi/python-practice/streamlit-app-deploy

3. アプリの起動:
   streamlit run app.py

4. ブラウザでアクセス:
   http://localhost:8501


【環境設定】

■必要な環境変数:
.envファイルにOpenAI APIキーを設定してください。

OPENAI_API_KEY=your-api-key-here

※ .envファイルは.gitignoreに含まれており、GitHubにはアップロードされません。
